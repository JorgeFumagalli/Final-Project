# Final Project

\# Previsão da Inadimplência de Cartões de Crédito no Brasil  

\*\*Projeto de Modelagem Preditiva – MBA USP/ESALQ em Data Science \& Analytics\*\*



Este repositório contém o pipeline completo de extração, preparação, modelagem e avaliação para previsão da inadimplência total de cartões de crédito no Brasil, utilizando séries temporais macroeconômicas oficiais.



O projeto compara algoritmos de Machine Learning e Deep Learning em dois cenários:



\- \*\*FULL\*\*: série completa (2015–2025)  

\- \*\*EXCL\*\*: série sem o período de instabilidade fiscal (2019–2021)
